## Driverscan

The script exports all third-party drivers detected in use (non-Microsoft).

Tested on:
```markdown
Windows 7 SP1
Windows 8.1
Windows 2012 R2
Windows 2016
Windows 10 Pro 1909
```

Example usage:
```markdown
.\driverscan.ps1

.\driverscan.ps1 -TargetDirectory "C:\temp\drivers"
```
